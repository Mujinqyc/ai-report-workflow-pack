# AI 日报周报自动生成工作流包 - 产品说明

## 1. 产品定位

这是一个面向职场办公场景的小型 AI 工作流包。

它帮助用户把零散的工作记录，整理成更适合发送、汇报和复盘的日报与周报内容。

## 2. 目标用户

- 需要每天写日报、周报的职场新人
- 需要向上级汇报工作进度的员工
- 想提高工作记录效率的个人用户
- 想学习 AI 办公自动化的小团队

## 3. 用户痛点

- 每天写日报容易重复、耗时
- 工作内容零散，不知道如何组织语言
- 汇报给领导时不知道如何表达重点
- 周报需要回顾一周内容，整理成本较高

## 4. 第一版解决方案

用户只需要填写三类信息：

- 今天完成的工作
- 遇到的问题
- 明天的计划

系统根据这些输入生成四种结果：

- 正式日报
- 简洁日报
- 领导汇报版
- 周报总结

## 5. 第一版交付内容

- 文档版：产品说明、使用指南、演示脚本
- 表格版：日报/周报输入模板
- 简单网页演示版：本地运行页面

## 6. 第一版边界

第一版先不做以下功能：

- 登录系统
- 支付系统
- 数据库
- 线上部署
- OpenAI API 接入

这些功能会在产品验证后，再逐步迭代。

## 7. 当前目标

先完成一个可以本地演示、可以作为作品集、可以继续迭代的小型 AI 数字产品 MVP。
