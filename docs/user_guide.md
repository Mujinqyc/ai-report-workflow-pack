# AI 日报周报自动生成工作流包 - 使用指南

## 1. 使用对象

这个工作流包适合需要写日报、周报或阶段性工作汇报的用户。

用户不需要懂编程，只需要按照模板填写工作内容。

## 2. 使用前准备

使用前先准备三类信息：

- 今天完成的工作
- 今天遇到的问题
- 明天计划做什么

如果要生成周报，还需要准备一周内每天的主要工作记录。

## 3. 日报生成流程

### 第一步：填写输入内容

在表格模板或网页演示页面中填写：

- 完成事项
- 遇到的问题
- 明日计划

### 第二步：选择输出类型

第一版支持四种输出：

- 正式日报
- 简洁日报
- 领导汇报版
- 周报总结

### 第三步：复制生成结果

生成结果后，可以复制到：

- 企业微信
- 钉钉
- 飞书
- 邮件
- 个人工作日志

## 4. 推荐填写方式

填写时尽量具体，不要只写很短的词。

不推荐：

- 写代码
- 开会
- 改问题

推荐：

- 完成日报生成页面的输入框布局
- 和同事确认周报模板字段
- 修复按钮点击后没有生成内容的问题

## 5. 示例输入

今天完成的工作：

- 完成网页页面结构
- 编写日报生成逻辑
- 整理产品说明文档

遇到的问题：

- 对 JavaScript 事件绑定还不熟悉
- 页面样式在不同宽度下显示不稳定

明天计划：

- 优化页面样式

## 6. 使用建议

- 每天工作结束前填写一次
- 每周五集中整理一周记录
- 不要追求一次写得完美，先保证信息完整
- 后续可以接入 AI API，让生成结果更自然
